RALSEI BOT

Because this fluffy boy needs to live on forever.

Introduction

Ralsei can automatically identify its name in normal conversations. Therefore, it is possible for Ralsei to respond when he is being talked about. If you wish, you may also summon Ralsei using the !ralsei command.

Commands

Command Description
Commands that are marked blue will be replied to using a text box with no face. Commands that are marked yellow will not be replied to using a text box.
!ralsei help Displays instructions on using Ralsei.
!ralsei <greeting> Make Ralsei say hello.

Example: !ralsei hello, !ralsei hi
!ralsei <farewell> Make Ralsei say goodbye.

Example: !ralsei goodbye, !ralsei farewell
!ralsei love Confess your feelings of admiration to Ralsei.
!ralsei hug <name> Hug a name. Does not work with usernames (yet.) Bonus points for special names.
!ralsei *hugs ralsei* Hug Ralsei. The greatest achievement possible.
!ralsei say <text> Make Ralsei say anything. Each box has 3 lines with 24 character slots.
!ralsei dbots Show Discord Bots statistics.
!ralsei ping Show Discord Bots statistics.
!ralsei ?????? Hidden command.
To access the chad, you must be glad.
The name for the ship that sails, are small yet bad...

Automatic Recognition

These messages will appear if Ralsei is used in a phrase. This commonly leads to a response. Any capitalization works in triggering these responses.

Trigger Description
Phrases that are marked blue will be replied to using a text box with no face. Phrases that are marked yellow will not be replied to using a text box.
<greeting> ralsei OR ralsei <greeting> Make Ralsei say hello.
<farewell> ralsei OR ralsei <farewell> Make Ralsei say goodbye.
i love you ralsei OR similar Confess your feelings of admiration to Ralsei.
thank you ralsei OR similar Give thanks to Ralsei.
hug me ralsei Make Ralsei hug you. Custom names are only accessible through the command.
*hugs ralsei* Hug Ralsei. The greatest achievement possible.
